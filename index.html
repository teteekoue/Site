À propos de moi

Bienvenue sur mon portfolio professionnel

Je suis TETE Ekoué Jean-François, un développeur passionné spécialisé dans la création d'applications web modernes et interactives. Mon parcours m'a permis de travailler sur des projets variés, allant des sites vitrines aux applications web complexes.

Je maîtrise les technologies frontend comme HTML, CSS, JavaScript, React, ainsi que les technologies backend comme Node.js, Python et les bases de données SQL/NoSQL.

Toujours à l'affût des nouvelles tendances technologiques, je m'efforce de créer des expériences utilisateur exceptionnelles tout en maintenant un code propre et maintenable.
